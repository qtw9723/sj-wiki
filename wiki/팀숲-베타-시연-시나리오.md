> [!tip] 핵심 takeaway
> 🧠 이 문서는 **교회 리더·대학부(비개발자)에게 보여주는 베타 시연용**이다. 기술 설명이 아니라 "한 대학생이 가입 → 성경 읽고 체크 → 나무 얻고 → 우리 팀 숲을 함께 키운다"는 **사용자 여정 스토리**로 짠다. 각 장면마다 ① 시연자 멘트(대본)와 ② 시연 전·중 점검 체크리스트를 함께 둔다.
> 🧠 [[내-프로필]] 관점: 이 시연 자료 자체가 "기획→개발→사용자 데모"까지 끌고 가는 포트폴리오 근거가 된다. 핵심 메시지는 **"진입장벽 0(이름+팀만으로 참여), 함께 자라는 재미"**.

> ⚠️ 출처 메모: 앱 흐름·기능은 [[팀숲-bible-forest]]와 설계 문서(📄)에서 가져왔다. 라이브에서 자동 확인된 건 `/register`(이름+팀 선택) 화면까지이며(로그인 이후 화면은 인증 필요로 미확인), **나머지 화면 명칭·버튼 위치는 시연 전에 실제 앱과 1회 대조 필요**(🧠 표시는 시연 연출·멘트 = Claude 작성).

## 1. 시연 전 준비 체크리스트 (시작 5분 전) 🧠

- [ ] 라이브 사이트가 정상 접속되는지 확인 (https://bible-forest-lake.vercel.app/)
- [ ] 시연용 **새 사용자**로 가입할 이름·팀 미리 정해두기 (예: 이름 "시연", 팀 "A팀")
- [ ] 비교용으로 **데이터가 이미 쌓인 팀/계정** 하나 준비 (랭킹·숲이 채워진 화면을 보여주기 위함)
- [ ] 나무가 1그루 이상 지급되도록 **체크할 성경 권/장 미리 정해두기** (10장 = 나무 1그루 규칙) 📄
- [ ] 화면 미러링/프로젝터 연결 + 글씨 크기(모바일 화면이면 확대) 확인
- [ ] 네트워크 확인 (모든 데이터가 API로 오므로 인터넷 필수) 📄
- [ ] 쿠키 로그인 60일 유지라 이전 세션이 남아있을 수 있음 → 깔끔한 시연 원하면 시크릿 창 사용 📄🧠

## 2. 본 시나리오 (장면별 대본 + 체크리스트)

### 장면 1. 진입장벽 0 — 가입 📄

**화면**: `/register` (팀 선택 + 닉네임)

**시연자 멘트 (대본)** 🧠
> "팀숲은 가입이 정말 간단해요. 회원가입, 비밀번호, 이메일 인증 그런 거 없습니다. 이름 적고, 우리 팀만 고르면 끝이에요. 대학부 누구나 30초 안에 시작할 수 있게 만들었습니다."

**동작**: 닉네임 입력(10자 이내) → 팀 선택 → [다음/시작] 클릭 📄

**점검 체크리스트**
- [ ] 닉네임 10자 초과 시 입력 제한되는가 (maxLength) 📄
- [ ] 팀 미선택 시 다음으로 못 넘어가는가 🧠
- [ ] 가입 즉시 메인 화면으로 진입하는가 📄
- [ ] (멘트 포인트) "비밀번호 없음 · 쿠키 60일 유지" 강조 📄

---

### 장면 2. 첫 화면 둘러보기 — 내 현황 한눈에 📄🧠

**화면**: 메인/대시보드

**시연자 멘트** 🧠
> "로그인하면 바로 내 상황이 보여요. 내가 신약 몇 장을 읽었는지, 나무는 몇 그루 모았는지, 우리 팀이 지금 몇 등인지. 복잡하지 않게, 한 화면에서 '나'와 '우리 팀'을 같이 볼 수 있게 했습니다."

**동작**: 대시보드의 내 진행률 / 보유 나무 / 팀 랭킹 영역을 차례로 가리키며 설명 🧠

**점검 체크리스트**
- [ ] 내 성경 진행률(읽은 장 수)이 표시되는가 📄
- [ ] 보유 나무 수가 표시되는가 📄
- [ ] 팀 랭킹/우리 팀 위치가 표시되는가 📄
- [ ] 숫자들이 API에서 실제로 불러와지는가(새로고침해도 유지) 📄

---

### 장면 3. 핵심 — 성경 읽고 체크 → 나무 획득 📄

**화면**: 성경 인증 페이지

**시연자 멘트** 🧠
> "이게 핵심이에요. 신약을 읽고 읽은 장을 체크합니다. 한 장 한 장 누를 때마다 저장되는 게 아니라, 다 고르고 마지막에 '완료'를 누르면 한 번에 기록돼요. 그리고 10장을 읽을 때마다 나무 한 그루를 받습니다. 신약 260장을 다 읽으면? 특별한 나무를 드려요."

**동작**: 한 권(book)을 펼쳐 여러 장 체크 → [완료] 클릭 → 나무 지급 안내 확인 📄

**점검 체크리스트**
- [ ] 개별 체크는 즉시 저장 안 되고, '완료' 클릭 시 **한 번에** 반영되는가 (권 단위 일괄 등록) 📄
- [ ] 10장 달성 시 나무가 지급되는가 (10장 = 1그루) 📄
- [ ] 260장 완독 시 특별 나무 규칙 (멘트로만 소개 가능) 📄
- [ ] 다시 들어와도 체크 상태가 유지되는가 📄
- [ ] (참고) 한 권을 다시 저장하면 그 권은 새 목록으로 교체됨(bulk replace) — 시연선 굳이 안 보여줘도 됨 📄🧠

---

### 장면 4. 내 나무 보기 — 모은 나무 확인 📄🧠

**화면**: 내 나무 보기(인벤토리)

**시연자 멘트** 🧠
> "받은 나무는 여기 내 보관함에 모여요. 아직 숲에 심지 않은 나무들이죠. 종류도 여러 가지라 모으는 재미가 있습니다."

**동작**: 인벤토리에서 보유 나무 목록 확인 📄

**점검 체크리스트**
- [ ] 장면 3에서 받은 나무가 인벤토리에 보이는가 📄
- [ ] 아직 심지 않은 나무 / 심은 나무 구분이 되는가 📄

---

### 장면 5. 함께 만드는 — 우리 팀 숲에 나무 심기 📄

**화면**: 우리팀 숲 (나무 배치)

**시연자 멘트** 🧠
> "이제 내 나무를 우리 팀 숲에 심습니다. 내가 원하는 자리에 놓을 수 있어요. 한 명 한 명이 심은 나무가 모여서 우리 팀만의 숲이 완성돼요. 혼자 읽는 게 아니라, 같이 키우는 겁니다."

**동작**: 인벤토리 나무 선택 → 팀 숲 격자에 배치(좌표 지정) → 저장 📄

**점검 체크리스트**
- [ ] 나무를 숲의 원하는 위치에 배치할 수 있는가 📄
- [ ] 배치 후 우리 팀 숲에 즉시 반영되는가 📄
- [ ] 새로 심은 나무가 기존 나무 위로 자연스럽게 보이는가(렌더링 순서) 📄
- [ ] (참고) 배치 좌표는 1회성(수정 불가) — 멘트에서 "신중히 심어요"로 풀어도 됨 📄🧠

---

### 장면 6. 동기부여 — 우리 팀 숲 전체 & 랭킹 📄🧠

**화면**: 우리팀 숲 전체 보기 + 팀 랭킹

**시연자 멘트** 🧠
> "팀 전체 숲을 보면 우리가 얼마나 읽었는지 한눈에 보여요. 그리고 팀별 랭킹! 다른 팀보다 우리 숲을 더 풍성하게 만들고 싶은 마음, 이게 끝까지 함께 읽게 하는 힘이라고 생각했어요."

**동작**: (미리 준비한 데이터가 찬 팀으로) 풍성한 숲 + 랭킹 화면 시연 🧠

**점검 체크리스트**
- [ ] 팀원들이 심은 나무가 한 숲에 모여 보이는가 📄
- [ ] 팀별 랭킹이 표시되는가 📄
- [ ] 데이터가 찬 팀과 빈 팀의 차이가 시각적으로 잘 드러나는가 🧠

---

### 장면 7. 소소한 재미 — 테마 선택 📄🧠

**화면**: 테마 선택 화면

**시연자 멘트** 🧠
> "마지막으로, 숲 테마를 골라서 분위기를 바꿀 수 있어요. 작은 요소지만 내 공간 같은 느낌을 줍니다."

**점검 체크리스트**
- [ ] 테마 선택/변경이 동작하는가 📄
- [ ] 선택한 테마가 숲 화면에 반영되는가 🧠

---

### 장면 8. (선택) 운영자 관점 — 챌린지 기간 안내 📄🧠

> 비개발자 대상이라 깊게는 안 가고, 한 문장으로만 소개 권장.

**시연자 멘트** 🧠
> "챌린지 기간이 정해져 있어서 기간 안에만 체크가 가능하고, 기간이 끝나도 우리가 만든 숲은 계속 볼 수 있어요. 수련회 때 대형 스크린으로 전시하고, 가장 열심히 키운 '베스트 가드너'를 시상할 계획입니다."

**점검 체크리스트**
- [ ] 기간 종료 후에도 조회는 되고 체크(쓰기)만 막히는 규칙 확인 📄

## 3. 마무리 멘트 🧠

> "팀숲은 '혼자 읽는 성경'을 '함께 키우는 숲'으로 바꾸는 앱이에요. 진입장벽을 0으로 만들어서 누구나 참여하고, 함께라서 끝까지 가게 만드는 게 목표입니다. 수련회 때 우리 대학부 전체의 숲이 어떻게 자라는지 같이 보면 좋겠어요. 써보시고 불편한 점 있으면 꼭 알려주세요!"

## 4. 예상 Q&A 🧠

| 질문 | 답변 |
|------|------|
| 비밀번호가 없으면 다른 사람이 내 걸로 들어오면? | 이름+팀으로 식별, 쿠키로 60일 유지. 베타라 단순화에 초점. 필요 시 보완 예정 📄🧠 |
| 잘못 체크하면 어떻게 하나요? | 권 단위로 다시 저장하면 그 권은 새로 고른 목록으로 교체됩니다 📄 |
| 읽은 걸 취소하면 나무도 사라지나요? | 네, 기준 미달이 되면 마지막에 받은 나무부터 회수돼요(배치한 나무 포함). "정직하게 읽기"를 위한 장치 📄 |
| 휴대폰에서도 되나요? | 모바일 화면 기준으로 만들었습니다(반응형) 📄🧠 |
| 기간 지나면 데이터 사라지나요? | 아니요. 조회는 계속 되고 기록은 남습니다. 체크(쓰기)만 기간 제한 📄 |

## 5. 시연 동선 요약 (한 장 컨닝페이퍼) 🧠

```
가입(이름+팀) → 대시보드(내 현황) → 성경 체크+완료(나무 획득)
→ 내 나무 보기 → 우리 팀 숲에 심기 → 팀 숲 전체+랭킹 → 테마 → 마무리
```
