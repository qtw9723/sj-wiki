> [!tip] 핵심 takeaway
> GLM은 중국 **Zhipu AI(Z.ai)**의 오픈소스 LLM 계열로, **"클로드급 코딩 성능을 1/5~1/10 가격에"**가 셀링포인트다. 너에게 의미 있는 두 갈래:
> - **① Claude Code 대체·병행용** → GLM Coding Plan **월 $18(Lite)**부터, Claude Code·Cline 등에 API 물려 쓴다. Anthropic 구독 대비 파격가라 사이드 프로젝트([[올림푸스-Olympus]]·[[개발-위시리스트]])의 토큰 비용 방어책.
> - **② LLM 앱 백엔드용** → [[Cogi-POC-Generator]]가 지금 OpenAI gpt-4o를 쓰는데, GLM API는 **입력 $0.6 / 출력 $2.2** (플래그십도 $1.4/$4.4)로 훨씬 싸다. **무료 모델(GLM-4.7-Flash 등)**도 있어 실험/POC 부담이 없다.
>
> 🧠 단, 중국 회사 모델이라 **회사(MWW) 데이터·사내 챗봇엔 데이터 거버넌스 확인 전까지 부적합**. 개인/포트폴리오·오픈소스 실험용으로 먼저 보는 게 맞다.

## 1. GLM이 뭔가 (한 줄)
- 📄 **Zhipu AI**(智谱, 칭화대 스핀오프)가 만드는 오픈소스 LLM 패밀리. 해외 브랜드는 **Z.ai**. "GLM = General Language Model".
- 📄 대표 강점: **코딩·에이전트·툴 사용**. GLM-4.6는 Claude Code·Cline·Roo Code·Kilo Code 등 20+ 코딩 툴에 바로 붙여 쓰도록 설계됨.
- 📄 라이선스: 주요 모델(GLM-4.5/4.6 등)이 **MIT 라이선스 오픈소스** → 상업적 사용·셀프호스팅 허용. HuggingFace(`zai-org/GLM-4.6`)·Ollama·OpenRouter로도 배포.

## 2. 모델 라인업 (2026-07 기준)
🧠 2025년 GLM-4.5/4.6에서 시작해 2026년 GLM-4.7 → GLM-5 → GLM-5.2로 빠르게 세대 교체 중. 현재 플래그십은 **GLM-5.2**.

| 모델 | 위치 | 비고 |
|---|---|---|
| **GLM-5.2** | 최신 플래그십 | API·Coding Plan 최상위. 📄 입력 $1.4 / 출력 $4.4 — "GPT-5.5의 약 1/6 가격" 홍보 |
| GLM-5 / GLM-5-Turbo | 상위 | Turbo는 속도 최적화 |
| **GLM-4.7** / 4.7-FlashX | 중급 | Flash 계열은 초저가~무료 |
| **GLM-4.6** | 코딩 주력(2025-09-30) | 📄 200K 컨텍스트, 357B MoE, MIT. 아래 §5 상세 |
| GLM-4.5 / 4.5-Air / 4.5-X | 이전 세대 | Air=경량 저가, X=고성능 고가 |
| **GLM-4.7-Flash / 4.5-Flash / 4.6V-Flash** | **무료** | 📄 입력·출력 전부 $0 — POC·실험용 |
| GLM-4.6V / 5V-Turbo / GLM-OCR | 비전·멀티모달 | 이미지·문서·OCR |

## 4. GLM Coding Plan (구독형, 코딩 툴용)
📄 API 종량제 대신 **정액 구독**으로 Claude Code·Cline 등에 붙여 쓰는 상품. 출처: `docs.z.ai/devpack` + aipricing.guru/felloai (2026-07-07).

| 티어 | 정가/월 | 30% 프로모 | 5시간당 프롬프트 | 주당 | MCP 호출/월 |
|---|---|---|---|---|---|
| **Lite** | **$18** | ~$12.6 | ~80 | ~400 | 100 |
| **Pro** | **$72** | ~$50.4 | ~400 | ~2,000 | 1,000 |
| **Max** | **$160** | ~$112 | ~1,600 | ~8,000 | 4,000 |

- 📄 전 티어 동일 모델: **GLM-5.2(플래그십) · GLM-5-Turbo · GLM-4.7 · GLM-4.5-Air**.
- 📄 **피크시간(14:00–18:00 UTC+8)**엔 GLM-5.2/5-Turbo가 쿼터를 3배(오프피크 2배) 소모. 단 ~2026-09 한정 프로모로 오프피크 1배.
- 📄 실사용량은 프로젝트 복잡도·레포 크기·auto-accept 여부에 따라 변동.
- ⚠ **정가는 자료마다 편차 있음**: 초기엔 $3/월 프로모(→2026-02-11 종료), 일부 자료는 분기결제(Lite $30/분기≈$10/월)로 표기. 실제 결제 전 **z.ai에서 현재가 재확인** 필요.

## 5. GLM-4.6 스펙 (코딩 주력 세대)
📄 `llm-stats.com/models/glm-4.6`
- **컨텍스트 200K** (GLM-4.5의 128K → 확장), **357B 파라미터 MoE**, **MIT 라이선스**, 출시 2025-09-30.
- 📄 8개 공개 벤치마크(에이전트·추론·코딩)에서 GLM-4.5 대비 전반 상승, DeepSeek-V3.1-Terminus·Claude Sonnet 4와 경쟁 우위 주장.
- 📄 GLM-4.5 대비 **작업당 토큰 ~15% 절감**하면서 복잡 멀티턴 성공률 개선 → 실사용 비용이 이중으로 저렴.
- ⚠ 멀티모달 표기 충돌: llm-stats는 "텍스트+이미지+비디오 입력"이라 하나, 공식 z.ai는 GLM-4.6를 **텍스트 전용**으로, 비전은 별도 **GLM-4.6V**로 분리해 둠. 이미지 처리는 4.6V 계열로 보는 게 안전.

## 6. 너에게의 적용 (🧠 판단)
- **사이드/오픈소스 프로젝트**: [[올림푸스-Olympus]]는 여러 Claude 세션이 병렬로 도는 구조라 토큰 소모가 크다. GLM Coding Plan(월 $18~)이나 GLM API를 백엔드 옵션으로 두면 실험 비용을 크게 낮출 수 있다. (단 올림푸스는 verify.sh 심판 구조라 모델 교체가 비교적 자유로움)
- **[[Cogi-POC-Generator]] 백엔드**: 현재 OpenAI gpt-4o. GLM API(또는 무료 Flash)로 A/B 해볼 가치. 생성 품질만 받쳐주면 운영비가 급감. ⚠ 단 Cogi는 사내 POC라 **회사 데이터 거버넌스**부터 확인.
- **회사(MWW) 사내 챗봇**: ⚠ 중국 회사 모델 → 사내 기밀·고객 데이터엔 신중. 셀프호스팅(MIT라 가능)으로 데이터를 외부로 안 내보내는 옵션은 있으나 GPU 비용·검토 필요. 이 판단·상세는 [[내-프로필]] 회사 맥락상 개인 vault엔 원칙만 남긴다.
- **Claude Code 병행**: 지금 주력은 Claude(Opus 4.8, [[Claude-Code-업데이트-동향]]). GLM은 "고빈도 저위험 코딩은 싼 모델, 어려운 건 Claude"식 이중 전략의 저가 축으로 유용.
