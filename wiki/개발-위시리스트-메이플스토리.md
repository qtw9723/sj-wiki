> 📄 **플랫폼 = 메이플스토리 월드(MSW)**, **언어 = Lua**(드래그앤드롭 병행), **메이플 IP·에셋 제공/활용 + AI 활용 허용**(공식이 **Claude Code**를 바이브 코딩 도구로 명시). 🧠 본인의 Claude Code·LLM·풀스택 경험이 그대로 강점. 심사는 **IP 해석·확장 40% / 완성도 40% / 지속가능성 20%**(아래 §콘테스트 요강).

## 개요
- 📄 **프로젝트명**: **글로벌 개발 콘테스트** (출품작 = 메이플스토리 관련 게임 개발)
- 📄 **영역**: 개인
- 📄 **상태**: 🟡 진행예정
- 📄 **🎯 출품 마감**: **2026-10-07** *(출품 기간 ~10/7)*
- 📄 **플랫폼**: **메이플스토리 월드(MapleStory Worlds)** — 넥슨의 UGC 샌드박스 플랫폼에서 개발·출품 *(콘테스트 필수 조건)*.
- 📄 **개발 언어**: **Lua**(확정 2026-07-02, 공식 문서). MSW는 **드래그앤드롭 제작 + Lua 스크립트** 병행 — 간단한 건 스크립트 없이, 심화 기능은 Lua로 구현. 공식 스크립트 에디터·API·클라이언트/서버 구조·이벤트 핸들러 제공, 난이도 Lv.1~4 가이드. ([Creator Center 루아 기초](https://maplestoryworlds-creators.nexon.com/ko/docs/?postId=822))
- 📄 **IP·저작권**: 메이플스토리 **IP·에셋은 콘테스트가 제공/활용 허용**, 단 출품작은 **타인의 저작권/IP를 침해하지 않는 원작**이어야 함. **AI 활용 허용**(공식). → 🧠 원작 세계관을 살리되 외부 무단 에셋은 배제. 심사 40%가 "IP 해석·확장"이라 이게 핵심 축.

## 콘테스트 요강 📄 (공식 페이지 확인 2026-07-02)
> 📄 출처 = 넥슨 공식 이벤트 페이지 [maplestoryworlds.nexon.com/events/ko/2026globalcontest](https://maplestoryworlds.nexon.com/events/ko/2026globalcontest). 아래는 이 공식 페이지 기준(넥슨 코리아 트랙).

- 📄 **명칭**: 메이플스토리 **글로벌 개발 콘테스트** — 총 지원금 **약 ₩18억** 규모.
- 📄 **출품 기간**: **2026.06.04 ~ 2026.10.07**.
- 📄 **참가 자격**: **만 14세 이상**(북미·유럽 거주자는 18세 이상), **개인 또는 팀** 참가 가능. 출품작은 **타인의 저작권/IP를 침해하지 않는 원작**이어야 하며, **AI 활용 허용**.
- 📄 **심사 기준(가중치)**: **IP 해석 및 확장 40% · 완성도 40% · 지속 가능성 20%**.
- 📄 **상금·지원금 구조**:
  - **본선 30팀**: 개발 지원 프로그램으로 팀당 **최대 약 $35,000(≈5,000만원)**.
  - **최종 수상 7팀**: **$10,000 즉시 지급** + 지급 기준 충족 시 **최대 $20,000 후속 개발 지원금** 추가 → **팀당 최대 $65,000(≈9,000만원)**.
- 📄 **일정**: **10/7** 출품 마감·1차 심사 시작 → **11/5** 본선(30팀) 발표·2차 시작 → **11/25** 2차 종료 → **12월 초** 최종 수상 발표.
- 📄 **플랫폼·개발**: 메이플스토리 월드 기반, **Lua** 구현. **"바이브 코딩" AI 지원 — Claude Code · Codex · Cursor** + **AI Toolkit** 기능 지원.
- 📄 **제공 리소스**: 오픈소스 코드·에셋이 포함된 **리메이크 월드 6종**(RPG·캐주얼/아케이드·전략·시뮬 장르).
- 📄 **지원 프로그램**: 장르별 핵심 시스템 교육자료, 전문가 Q&A 창구, 전용 **디스코드** 커뮤니티, **원데이 부트캠프**.
- 🧠 함의(내 관점): **AI 활용 허용 + 공식이 Claude Code를 바이브 코딩 도구로 명시** → [[내-프로필]]의 Claude Code·LLM·풀스택 경험이 그대로 강점. 심사 40%가 **완성도**, 40%가 **IP 해석·확장**, 20%가 **지속 가능성**이므로 **원작 세계관을 확장하는 뾰족한 컨셉 + 완성도 + 반복/운영 가능한 구조**에 집중하는 게 유리. 마감까지 약 3개월.
